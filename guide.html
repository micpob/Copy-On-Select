Welcome to Copy on Select!

How it works:

CAVEATS:

  • The extension does not work on any page whose URL starts with the word "chrome", for example chrome://extensions/ or chrome://settings/ or this page (this is standard behaviour for all Chrome extensions)
  • The extension does not work on the page that opens when you click on the + sign to open a new tab in Chrome. Usually this is the google homepage (same as above, this is standard for all Chrome extensions)
  • The extension does not work if you select some text in the browser's URL bar (because it is outside of the browsing window)

COPY:

  • Select some text and it will get copied automatically.
  • If you want to bypass the auto-copying while selecting something, hold down the Ctrl key (Command on Mac).
  • You will see a short 'Copied!' alert every time you select something. You can deactivate it in the Options page if you don't like it.

PASTE:

Method 1: Using double click.

  • Double click on a input field while holding down the Ctrl key (Command on Mac) and the copied text will be automatically pasted.
  • Hold down the Shift key while double clicking and the field will be emptied of any pre-existing content before pasting.
  • You can deactivate any of this functionalities in the Options page.

Method 2: Using middle click.

  • Do a middle click (mouse wheel) on a input field and the copied text will be automatically pasted.
  • Hold down the Shift key while middle clicking and the field will be emptied of any pre-existing content before pasting.
  • If you want to bypass the auto-pasting on middle click, hold down the Ctrl key (Command on Mac).
  • You can deactivate any of this functionalities in the Options page.
